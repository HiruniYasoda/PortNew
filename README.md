# 🚀 Hiruni Yasoda | Personal Portfolio

A sleek, modern, and highly interactive **personal portfolio website** designed to showcase my journey as a **Full-Stack Developer** transitioning into **Cloud, DevOps, and Technical Project Management**.

Built with performance and aesthetics in mind, this portfolio features a **dark theme with neon purple accents**, **glassmorphism UI elements**, and **smooth, complex animations**.

---

## ✨ Features

- **Modern UI/UX** – Dark mode with neon gradients and glassmorphism effects  
- **Smooth Animations** – Scroll-triggered and interactive animations using Framer Motion  
- **Bento Grid Layout** – Clean and responsive navigation layout  
- **Fully Responsive** – Optimized for mobile, tablet, and desktop  
- **Fast & Optimized** – Powered by Vite for fast builds and HMR  

---

## 🛠️ Tech Stack

- React.js  
- Vite  
- Tailwind CSS  
- Framer Motion  
- Lucide React  

---

## 🚀 Getting Started

Follow the steps below to run this project locally on your machine.

---

## 📋 Prerequisites

Ensure you have the following installed:

- Node.js  
- npm  

---

## 📥 Installation

### 1️⃣ Clone the repository
```bash
git clone https://github.com/HiruniYasoda/PortNew.git
 ```
### 2️⃣ Navigate to the project directory
```bash
cd PortNew
 ```
### 3️⃣ Install the dependencies
```bash
npm install
```
### ▶️ Start the Development Server
```bash
npm run dev
```
### The application will be available at:
```bash
(http://localhost:5173)
```
### 📦 Building for Production
```bash
npm run build
```
